# Changelog

All notable changes to PortfolioLens will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-25

### Added
- Initial release of PortfolioLens
- Multiple portfolio support with custom names
- Manual holding entry (ticker, shares, cost basis, current price)
- Portfolio overview with total value and gain/loss calculations
- Asset allocation pie chart by sector
- Individual holding detail views with performance metrics
- Manual price update functionality
- Performance tracking with snapshot system
- Performance chart visualization over time
- Dividend payment tracking and logging
- Annual dividend income and yield calculations
- Asset type and sector classification
- CSV export for portfolios
- Dark mode support
- Universal app support (iPhone, iPad, Mac via Catalyst)
- SwiftData local persistence
- Swift Charts integration for visualizations
- MVVM architecture
- No external dependencies
- Privacy-focused local-only storage

### Technical Details
- Minimum iOS 17.0, iPadOS 17.0, macOS 14.0
